# Y2K38-ASEAN Cyber-System | Huokaing Thara

## Overview

This interactive web application serves as an **educational cybersecurity hologram interface** highlighting the imminent [Year 2038 problem (Y2K38)](https://en.wikipedia.org/wiki/Year_2038_problem) and raising awareness within an ASEAN multi-language context.

Through dynamic text-to-speech, mysterious hologram effects, binary rain animation, and live countdown, the app blends cybersecurity education with futuristic UI design.

---

## Features

- **Multi-language support** for 11 ASEAN languages including English, Khmer, Thai, Vietnamese, Indonesian, Malay, Filipino, Lao, Bengali, Burmese, and Tetum.
- Interactive **cybersecurity audit tools** showing cost estimations tailored by country multipliers.
- **Live countdown timer** until the UNIX 32-bit integer time overflow (Jan 19, 2038 UTC).
- Real-time **estimated cost to fix the Y2K38 problem per 24 hours**, dynamically adjusting with the countdown.
- **Speech synthesis** assists visually impaired users and gives audible feedback.
- **Mysterious hologram effect** with glowing borders and subtle animations for immersive experience.
- Classic **binary rain animation** background for aesthetic and thematic appeal.
- Background **ambient music** with mute/unmute controls.
- Security precautions: disables F12, right-click, and common inspect shortcuts to protect code integrity.

---

## Educational Purpose

This tool is designed to:

- Demonstrate the impact and urgency of the **Y2K38 time overflow problem**.
- Illustrate cybersecurity concepts via interactive and multilingual audit steps.
- Showcase how text-to-speech and web animations can enhance educational web interfaces.
- Foster awareness about timely cyber infrastructure updates required in the ASEAN region and beyond.

---

## Usage

1. Select a language from the top bar to change displayed text and speech language.
2. Choose a country to apply a multiplier affecting audit cost calculations.
3. Hover over the cybersecurity steps to see detailed descriptions.
4. Click on any audit step to:
    - Hear its estimated cost via speech.
    - Open a penetration testing tool reference page in a new tab.
5. Monitor the live countdown showing time until the Y2K38 overflow event.
6. Observe the estimated daily cost to fix the Y2K38 problem, updating in real-time.
7. Use the mute/unmute button to toggle background music.
